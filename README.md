# androidUIAutomation-todoAPP-FirstTry
Percobaan membuat script automation untuk mobile UI (Android) menggunakan Appium dan Java (First Try).

Dependencies:
- Appium
- Junit5

Tools :
- Appium server
- Appium inspector (untuk mencari id suatu element pada ui aplikasi)
